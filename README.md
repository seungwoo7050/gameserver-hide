# DungeonHub

DungeonHub is a C++ server project blueprint aimed at Neople-style action RPG backend roles.
It focuses on session management, instance lifecycle, and data integrity for dungeon-based gameplay.

## Build
```bash
cmake -S . -B build
cmake --build build
```

## Run
```bash
./build/dungeonhub
```
